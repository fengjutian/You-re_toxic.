
---

## **1️⃣ 人格与性格测试**

| 测试                               | 简介                             | 适用             |
| -------------------------------- | ------------------------------ | -------------- |
| **MBTI（Myers–Briggs）**           | 四个维度16种人格类型                    | 职业规划、团队沟通      |
| **大五人格（Big Five / OCEAN）**       | 外向、责任心、宜人性、神经质、开放性             | 科学研究、心理咨询、人格评估 |
| **DISC**                         | 支配型(D)、影响型(I)、稳定型(S)、谨慎型(C)    | 销售、管理、团队建设     |
| **霍兰德职业兴趣测试（RIASEC）**            | 6类兴趣类型：现实型、研究型、艺术型、社会型、企业型、常规型 | 职业规划、求职        |
| **16PF（16 Personality Factors）** | 16个人格维度                        | 人力资源、职业发展      |

---

## **2️⃣ 智力与能力测试**

| 测试                                     | 简介                    | 适用        |
| -------------------------------------- | --------------------- | --------- |
| **WAIS / WISC（韦氏智力量表）**                | 成人/儿童智商测试             | 教育评估、心理咨询 |
| **Raven’s Progressive Matrices（瑞文测验）** | 非语言智力测试，抽象推理          | 学术研究、招聘测评 |
| **多元智力测试（Gardner）**                    | 语言、逻辑、空间、音乐、肢体、社交、内省等 | 教育、学习风格分析 |

---

## **3️⃣ 情绪与心理健康测试**

| 测试              | 简介            | 适用          |
| --------------- | ------------- | ----------- |
| **抑郁量表（PHQ-9）** | 评估抑郁症状程度      | 临床、心理咨询     |
| **焦虑量表（GAD-7）** | 评估广泛性焦虑症状     | 心理健康监测      |
| **压力量表（PSS）**   | 评估生活压力水平      | 心理健康、职业压力管理 |
| **情绪智力量表（EQ）**  | 测量自我情绪管理和社交能力 | 职业发展、团队合作   |

---

## **4️⃣ 职业与兴趣测试**

| 测试                  | 简介           | 适用        |
| ------------------- | ------------ | --------- |
| **霍兰德职业兴趣（RIASEC）** | 六种兴趣类型       | 职业指导      |
| **职业性格倾向测评（如职业锚）**  | 评估工作动机和职业偏好  | 职业规划、岗位匹配 |
| **价值观与动机测试**        | 测量工作价值观和激励因素 | HR、管理     |

---

## **5️⃣ 其他心理测评**

* **人格障碍自测（如PDQ-4+）**：用于初步筛查人格特质异常
* **儿童心理发展量表**：智力、情绪、社交发展评估
* **恋爱/情感心理测试**：爱情风格、依恋类型（安全型、回避型、焦虑型）
* **睡眠与生活习惯测试**：评估睡眠质量和压力负荷

---

💡 **总结：**

* **MBTI、DISC、大五人格** → 人格类型
* **智力测验** → 逻辑、抽象、认知能力
* **情绪/心理健康量表** → 抑郁、焦虑、压力
* **职业兴趣测试** → 职业规划、岗位匹配
